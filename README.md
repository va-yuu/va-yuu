# Hi there! 

## Qui suis-je ?

Je suis **Vayu**, développeur spécialisé dans la création de bots Discord en JavaScript. Je conçois des solutions d'automatisation, des systèmes de modération avancés pour améliorer l'expérience des communautés Discord et bien d'autre.

## Projets actuels

Je travaille principalement sur :

* Développement de bots Discord personnalisés
* Systèmes de modération, sanctions automatisés
* Intégration d'APIs externes (Spotify, YouTube, Twitch, Minecraft, etc.)
* Optimisation des performances et gestion de bases de données

## Stack technique

* **Backend :** Node.js, Discord.js v14
* **Bases de données :** SQLite
* **Outils :** Git, VS Code, PM2
* **APIs :** REST, WebSocket
